## Sankey

The Sankey component is the main component that every Sankey diagram needs to have. It acts as a wrapper for the whole diagram, providing the essential structure and context for all other components like nodes and links. This component manages the overall layout to render the svg paths.

### Usage

import ImportTabs from '../../../components/ImportTabs.astro';

<ImportTabs componentName="Sankey"/>

### Properties

#### minpathheight

| | |
|---|---|
| Description | Sets the minimum height for paths in the diagram in pixel. Svelte-sankey automatically scales the visualization of each item and link based on the minpathheight and maxpathheight |
| Type | `number` |
