import { Tabs, TabItem } from '@astrojs/starlight/components';

Get started with svelte-sankey by installing it in your Svelte project. Choose your preferred package manager from the tabs below.

## Install svelte-sankey

<Tabs syncKey="package-managers">
<TabItem label="npm">

```bash
npm install svelte-sankey
```

</TabItem>
<TabItem label="pnpm">

```bash
pnpm add svelte-sankey
```

</TabItem>
<TabItem label="yarn">

```bash
yarn add svelte-sankey
```

</TabItem>
</Tabs>

## Next steps

After installation, you're ready to start creating beautiful Sankey diagrams. Check out the [component documentation](../../components/sankey) to learn about the available components and their properties.
