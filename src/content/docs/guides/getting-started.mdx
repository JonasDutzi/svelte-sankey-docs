## Basic Usage

After installation, you're ready to start creating beautiful Sankey diagrams.

### Data Structure
To create a Sankey diagram, you need to structure your data in a specific format. The data consists of four main hierarchical levels:

**columns** --> **rows** --> **items** --> **links**

- **columns**: Represent the vertical sections of the Sankey diagram.
- **rows**: Each column contains at least one row. You can use rows to group items within a column.
- **items**: Each row contains one or more items. Items are the nodes in the Sankey diagram.
- **links**: 
Each item has links to other items in subsequent columns. The link between items is determined by the **id** field of each item.

Below is an example of a simple data structure for a sankey diagram. It demonstrates how to structure your data and utilize the main components mentioned above.

				items: [
					{
						id: "item1",
						label: "Item 1",
						links: [
							{
								target: "item4",
								value: 5
							}
						]
					},
					{
						id: "item2",
						label: "Item 2",
						links: [

					}
				]
			},
			{
				rowLabel: "Row 2",

						links: [
							{
								target: "item4",
								value: 8
							}
						]
					}
				]
			}
		]
	},
	{
		id: "column2",
		columnLabel: "Column 2",
		rows: [
			{
				rowLabel: "Row 1",
				items: [
					{

   ```svelte
    <script lang="ts">
        import { Sankey, ColumnContent, Item, Anchor, AnchorContent } from 'svelte-sankey';
        import type { SankeyData } from 'svelte-sankey';
        import { sankeyData } from './sankeyData.ts';
        
    </script>
    
    <Sankey 
        minpathheight={1}
        maxpathheight={32}
        onPathClick={(event) => console.log("path clicked", event)}
        {#each sankeyData as column, columnIndex}
            <ColumnContent {column} ariaLabel={colLabel}>
                <div>
                    {column.columnLabel}
                </div>
                {#each column.rows as row, rowIndex}
                    <div>
                        <div>{row.rowLabel}</div>
                        {#each row.items as item}
                            <Item {item}>
                                    <Anchor {item} />
                                    <AnchorContent
                                        {item}
                                        onItemClick={(event) => console.log("item clicked", event)}
                                    >
                                        <div>
                                            {item.label}
                                        </div>
                                    </AnchorContent>
                            </Item>
                        {/each}
                    </div>
                {/each}
            </ColumnContent>
        {/each}
    </Sankey>
    ```
